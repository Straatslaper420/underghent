Two commands worth knowing:

Check if Aider has work in frontline/ not yet promoted to live:
--------
git diff --no-index index.html frontline\index.html
--------
(No output = identical/in sync. Output = frontline is ahead, ready to promote.)

If you ever want to promote it yourself (the copy-paste, automated):
-------------
Copy-Item frontline\index.html index.html
git add index.html
git commit -m "Promote frontline changes to live root index.html"
git push
---------------
